# Revolut Converter [Demo]

The app allows converting currency rates using base currency - touch any rate to make it 'base'.
Rates being downloading every 1 SEC, last value cached for temporary offline mode.

This project based on:
* Revolut currency API
* Kotlin
* RxJava 2
* Retrofit 2
* Dagger 2
* JUnit, Mockito
* AndroidX (ViewModel, LifeCycle, ConstraintLayout, RecyclerView, SwipeRefreshLayout, etc)
